# USAGE.md — 项目使用说明

> 本文档详细介绍智慧共享财务平台等系统的功能概述、使用方法与操作规范。面向最终用户编写，也可供 AI 理解业务场景。

---

## 一、系统概述

智慧共享财务平台是面向企业的财务管理 SaaS 系统，提供发票管理、合同管理、费用报销、预算管控、财务报表等一站式财务数字化能力。系统支持 PC Web 与移动端 H5 双端访问，数据实时同步。

### 角色与权限

| 角色 | 典型用户 | 主要能力 |
|------|----------|----------|
| 超级管理员 | 财务负责人、IT 管理员 | 全部功能、系统配置、用户管理 |
| 部门管理员 | 部门财务对接人 | 本部门发票 / 合同 / 报销审批 |
| 普通用户 | 业务人员 | 提交报销、查看个人相关财务数据 |

## 二、访问方式

### 2.1 PC 端

- 访问地址：`https://finance.company.com`
- 浏览器：Chrome / Edge 最新版（不支持 IE）
- 推荐分辨率：≥ 1366 × 768

### 2.2 移动端

- 访问地址：`https://finance.company.com/m`
- 或扫描 PC 端「移动端入口」二维码

### 2.3 桌面客户端（Electron）

- 适用于 Windows / macOS，支持自动更新。
- 启动后自动检查更新，强制更新版本须更新后才能使用。

## 三、发票管理操作

### 3.1 录入发票

1. 进入「发票管理 → 发票列表」，点击「新增发票」。
2. 选择录入方式：
   - **手动录入**：填写发票代码、号码、开票日期、金额、税额。
   - **OCR 识别**：上传发票图片，系统自动提取字段，可二次校对编辑。
3. 点击「保存」，发票进入「待审核」状态。

### 3.2 审核发票

1. 部门管理员在发票列表筛选「待审核」。
2. 点击「审核」，核对信息后选择「通过」或「驳回」。
3. 驳回需填写原因，提交人会收到通知。

> **注意**：同代码+号码的发票不可重复录入，系统自动查重拦截。

## 四、合同管理操作

### 4.1 登记合同

1. 进入「合同管理」，点击「新增合同」。
2. 填写合同名称、相对方、金额、起止日期，上传合同 PDF。
3. 保存后合同状态为「待签署」。

### 4.2 发起签署

1. 在合同详情页点击「发起签署」。
2. 系统对接 ECP 电子合同平台，生成签署链接。
3. 签署完成后状态自动更新为「已签署」。

### 4.3 到期提醒

- 合同到期前 30 天 / 7 天系统自动发送站内信 + 邮件提醒。

---

## 五、费用报销操作

### 5.1 提交报销单

1. 进入「费用报销 → 我的报销单」，点击「新建报销单」。
2. 填写报销事由、选择费用类型、关联已审核发票。
3. 上传附件（如出差审批单），提交审批。

### 5.2 审批流程

- 单笔 < 10000 元：部门管理员审批 → 财务终审。
- 单笔 ≥ 10000 元：部门管理员审批 → 财务终审（需双签）。
- 审批人可「同意」「驳回」「转签」「加签」。

## 六、财务报表操作

### 6.1 查看报表

1. 进入「财务报表」，选择报表类型（收支 / 发票统计 / 部门费用）。
2. 选择时间范围与筛选条件，点击「查询」。
3. 支持图表与表格双视图切换。

### 6.2 导出报表

- 点击「导出」，选择 Excel 或 PDF 格式。

## 七、系统管理操作（仅超级管理员）

### 7.1 用户与部门管理

- 「系统管理 → 用户管理」：新增用户、分配角色、调整所属部门。
- 「系统管理 → 部门管理」：维护部门树结构。

### 7.2 客户端包管理

1. 进入「系统管理 → 包管理」。
2. 上传新版本安装包（exe），系统自动生成 yml 并存入 MinIO。
3. 配置发布策略：
   - **强制更新**：勾选后，低于该版本的客户端必须更新。
   - **灰度发布**：设置灰度比例（如 10%），按用户 ID 哈希命中推送。
4. 点击「发布」，客户端下次检查更新时生效。
5. 支持版本回滚：将「当前生效版本」切回历史版本。

> 包存储与更新机制详见 [INTEGRATIONS.md](./INTEGRATIONS.md) 对象存储章节，功能边界详见 [FEATURES.md](./FEATURES.md) 客户端更新模块。

## 八、常见问题

| 问题 | 解决方案 |
|------|----------|
| 忘记密码 | 在登录页点「忘记密码」，通过绑定邮箱重置 |
| 发票 OCR 识别失败 | 检查图片清晰度，或切换手动录入 |
| 客户端无法启动 | 检查是否为强制更新版本，更新后重试 |
| 报销提交被拦截 | 检查关联发票是否已审核、金额是否超预算 |
| 导出报表无响应 | 大数据量导出为异步，请在「通知中心」查看进度 |

## 九、操作规范与安全须知

- **账号安全**：禁止共享账号，每人独立账号，离职及时停用。
- **数据保密**：财务数据属公司机密，禁止导出后外传。
- **审批时效**：审批人应在 2 个工作日内处理，超时系统自动催办。
- **操作留痕**：所有关键操作记录审计日志，可追溯（见 [CONCERNS.md](./CONCERNS.md) 审计日志）。
